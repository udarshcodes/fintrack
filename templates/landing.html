About FinTrack.

Built to Learn, Designed to Scale

FinTrack is a full-stack web application designed to help users take back control of their finances. Originally built as a final project for Harvard's CS50x course, it has evolved into a fully functional expense and budget tracker featuring a premium, Apple-style glassmorphic interface.

Under the hood, it uses Python, Flask, and an SQLite database to ensure your data is processed efficiently and stored securely. Experience real-time validations, dynamic monthly budgeting, and seamless history tracking—all wrapped in a beautifully responsive design.
